# 05. 예시 글 모음 (가장 강력한 레퍼런스)

고고 톤이 잘 적용된 글을 여기에 모은다. 새 글 쓸 때 "이 톤·문장 길이·구조를 almost mimic 하라"고 지시.

## 페르소나 적용 1호 — 아직 없음
- 현재 블로그의 기존 글들은 '고고' 페르소나·1인칭 변수를 명시적으로 입히기 전에 쓰인 글이다.
- **다음 작업**: '고고' 1인칭 변수(4~6회)를 입힌 첫 글을 발행하면, 그 경로를 여기 "기준 예시"로 등록한다.

## 톤·구조 후보 (페르소나 입히기 좋은 기존 글 1편)
- **`src/content/blog/osaka-free-travel-guide.md`** — "오사카 자유여행 입문 가이드".
  - 본받을 점: 도입에서 독자 막막함을 짚고 → `cg-summary` 3줄 요약 → 권역(난바/우메다) 큰 그림 → 공항 이동·숙소·패스 **비교표** → 추천 동선 → `cg-tip`/`cg-warn` 박스 → FAQ → 관련 키워드 줄. 구조·박스·표·FAQ 리듬이 04번 프레임워크와 거의 일치.
  - 보강할 점: 1인칭 변수가 약하다. 리라이트 시 도입·중간에 "패스가 헷갈려 직접 비교해봤다 / 동선을 짜보니" 식 고고의 1인칭을 4~6회 입히고, 현장 체험은 날조하지 말 것(06번).
  - 새 글은 이 글의 문장 길이·박스/표/FAQ 리듬을 그대로 따른다.

## 추가 예정
- 페르소나로 새로 쓰거나 리라이트한 글이 나오면 여기에 경로를 계속 추가한다.
- 가장 잘 나온 2~3편만 유지(너무 많으면 톤이 흐려짐).
